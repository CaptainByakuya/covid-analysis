# 🌍 COVID-19 Global Data Analysis

## Overview
A global analysis of COVID-19 spread, death rates and vaccination progress 
across 243 countries from January 2020 to October 2023 using Python and 
data visualization libraries.

## Questions Explored
- Which countries had the most cases and deaths?
- How did COVID-19 spread globally over time?
- How did the global vaccination rollout progress?
- Did a country's wealth (GDP) affect its death rate?

## Key Insights
- The USA led both total cases (103M) and deaths (1.1M)
- China's low death count raises questions about underreporting
- The Omicron wave in early 2022 was the largest global spike
- Vaccination rollout accelerated rapidly through 2021
- Wealth alone did not determine COVID-19 outcomes

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly

## Dataset
[Our World in Data COVID-19 Dataset](https://www.kaggle.com/datasets/caesarmario/our-world-in-data-covid19-dataset)
